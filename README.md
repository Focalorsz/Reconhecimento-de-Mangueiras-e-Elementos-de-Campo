# Reconhecimento-de-Mangueiras-e-Elementos-de-Campo
Comparativos de performance: segmentação por cor/forma (abordagem heurística) ou detecção por Rede Neural.

**Passo a Passo Execução**
1. **Clone o repositório**
```bash
git clone https://github.com/Focalorsz/Reconhecimento-de-Mangueiras-e-Elementos-de-Campo.git
cd main
```
2. **Crie um ambiente virtual (recomendado)**
```bash
python -m venv venv
# No Linux/Mac:
source venv/bin/activate
# No Windows:
venv\Scripts\activate
```
3. **Instale as dependências**
```bash
pip install -r requirements.txt
```
4. **Execute a aplicação**
```bash
python *um dos arquivos da main, segmentação por cor, modelo ou hibrido
```
